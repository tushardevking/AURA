# AURA — Autonomous Unified Research & Analytics Agent

An agentic AI analytics copilot built with LangGraph.

## What it does
- Upload any CSV dataset
- Ask questions in natural language
- AI agent writes and executes pandas code automatically
- Generates charts and consulting-style insights

## Stack
- LangGraph — agent orchestration
- Groq API + LLaMA 3.3 70B — LLM
- Pandas — data analysis
- Plotly — visualization
- FastAPI — backend (coming soon)
- Streamlit — UI (coming soon)
